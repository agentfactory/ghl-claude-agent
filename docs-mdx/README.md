# Claude Academy: GHL Agent Guide

Documentation site for connecting Claude to GoHighLevel using MCP. Built with Mintlify. Written for GHL agency owners with no developer background.

The guide walks through creating a Private Integration Token, connecting Claude, and running your first live test. It includes four ready-to-use prompt templates for common GHL workflows.

This site is the entry point for the Claude Academy community on Skool. GHL users share prompts, ask questions, and access the paid Agent Pack there.
