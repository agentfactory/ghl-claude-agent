### 4 Pre-Built Agent Configurations

Ready-to-paste system prompts for the four most common GHL workflows. Each one turns Claude into a specialized agent for a specific job:

| Agent | What It Does |
|-------|-------------|
| **Lead Qualifier** | Pulls new contacts, assesses them against your criteria, tags them Hot/Warm/Cold, and adds summary notes |
| **Follow-Up Writer** | Reads conversation history and drafts personalized messages in your voice |
| **Pipeline Manager** | Reviews your pipeline stages, finds stalled deals, moves them, and generates reports |
| **Appointment Setter** | Checks calendar availability, books meetings, and handles rescheduling |

Each configuration includes placeholder variables you fill in once. After that, the agent works with your specific account, your tags, your pipeline names, and your preferences.

A prompt library organized by GHL function:

- **Contacts:** Create, search, tag, update, and merge contacts (5 prompts)
- **Conversations:** Summarize threads, draft replies, flag urgent, archive old (4 prompts)
- **Pipeline:** Move stages, list stalled, generate reports, bulk tag (4 prompts)
- **Calendar:** Check availability, book, reschedule, cancel (4 prompts)
- **Reporting:** Weekly pipeline summary, lead source breakdown, team activity (3 prompts)

Every prompt includes placeholders and a description of what Claude will return.

A single-page checklist covering:

- Setup steps (token, Location ID, connection)
- Agent deployment (paste the config, name it, test it)

Print it, check it off, and you are running.
