# Connect Claude to Your GoHighLevel Account

This guide walks you through connecting Claude directly to GoHighLevel using MCP (Model Context Protocol). When you finish, Claude will be able to read, write, and manage your GHL account using plain English.

No third-party tools. No monthly middleware fees. No code to write.

## Who This Guide Is For

You run a business or agency on GoHighLevel. You want Claude to handle repetitive CRM tasks for you. You have zero interest in becoming a developer to make that happen.

That is exactly who this was built for.

## What You Will Be Able to Do

After completing this guide, you can ask Claude things like:

- "Pull up the contact info for John Smith and summarize his activity."
- "Write a follow-up message for every lead that went cold this week."
- "Move all stalled deals in my Sales Pipeline to the Needs Attention stage."
- "Check my calendar and book an appointment for tomorrow at 2pm."

Claude handles it directly inside your GHL account. No switching tabs. No copy-pasting between tools.
