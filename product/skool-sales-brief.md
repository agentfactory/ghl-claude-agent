## FAQ

**Q: Do I need any technical skills to use this?**
A: No. If you can copy and paste text, you can deploy these agents. The checklist walks you through every step with screenshots and confirmation checks.

**Q: Does this work with Claude Desktop or Claude Code?**
A: Both. The agent configurations and prompts work the same way in either version. The setup guide covers both connection methods.

**Q: What GHL plan do I need?**
A: You need a GHL plan that includes API access. This is typically the Pro plan or above. If you can see "Private Integrations" in your GHL Settings, you are good to go.

**Q: Will this send messages to my contacts without my permission?**
A: No. Every agent is configured to show you drafts and ask for approval before sending anything. You control what gets sent. The safety rules are built into every system prompt.

**Q: What happens when GHL adds new MCP tools?**
A: We update the prompt library and agent configurations to cover new capabilities. You get every update included in your purchase. No recurring fee.
