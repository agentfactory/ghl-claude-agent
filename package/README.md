# Claude Academy Agent Pack

Thank you for purchasing the Agent Pack. This package contains everything you need to connect Claude to your GoHighLevel account and start running pre-built agent workflows.

## What Is Inside

| File | Description |
|---|---|
| `quickstart-checklist.md` | Step-by-step setup checklist from first login to live agent |
| `agent-configs.md` | 4 ready-to-paste agent system prompts (Lead Qualifier, Follow-Up Writer, Pipeline Manager, Appointment Setter) |
| `prompt-library.md` | 20 copy-and-paste prompts organized by GHL function (Contacts, Conversations, Pipeline, Calendar, Reporting) |

## Where to Start

1. Open `quickstart-checklist.md` and work through it top to bottom. It covers account prerequisites, Private Integration Token setup, connection testing, and your first agent deployment.

2. Once connected, open `agent-configs.md` and deploy your first agent. We recommend starting with the Lead Qualifier.

3. Use `prompt-library.md` for additional day-to-day commands you can run through Claude at any time.
