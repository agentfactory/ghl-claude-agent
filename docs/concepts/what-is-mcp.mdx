# What is MCP?

MCP stands for Model Context Protocol. Here is what that means in plain English.

## The Simple Version

Think of MCP as a universal adapter. The same way a USB cable lets your phone talk to your computer, MCP lets Claude talk to GoHighLevel.

Before MCP, getting Claude to interact with your GHL account required:

- Copy-pasting data between tabs
- Setting up Zapier or Make workflows
- Hiring a developer to build custom integrations
- Monthly fees for middleware tools

With MCP, Claude connects directly. No middleman. No monthly fees. No code.

## How It Works (Without the Jargon)

Here is the flow when you ask Claude to do something in GHL:

```
You type a request in Claude
        |
Claude understands what you need
        |
Claude uses MCP to talk directly to GHL
        |
GHL processes the request and sends back results
        |
Claude shows you the results in plain English
```

You never see the technical layer. You just type what you want and Claude does it.

## A Real Example

**You type:** "Find the contact named Sarah Johnson and tell me her last conversation."

**What happens behind the scenes:**

1. Claude uses the MCP connection to search your GHL contacts
2. GHL finds Sarah Johnson and returns her contact record
3. Claude pulls her conversation history through the same connection
4. Claude reads through the messages and gives you a summary

**What you see:** A plain English summary of your last conversation with Sarah.

The whole thing takes a few seconds.

## Why This Matters for GHL Users

GHL is powerful, but it has a lot of screens and menus. MCP lets you skip the clicking and just tell Claude what you need done.

Instead of navigating to Contacts, searching, clicking the profile, scrolling to conversations, and reading through messages, you just ask.

This is not about replacing GHL. It is about having a capable assistant that already knows how to use every feature in your account.

## Is This Official?

Yes. GoHighLevel built MCP support directly into their platform. This is not a hack or workaround. GHL provides an official MCP endpoint that Claude connects to.

The endpoint is: `https://services.leadconnectorhq.com/mcp/`

You do not need to install or host anything to use it.
