What People Ask

Do I need to be technical to use this?
No. If you completed the free setup guide, you have all the skills you need. The agent configs are copy-paste. The prompts are fill-in-the-blank.

Does this work with any GHL account?
Yes. It works with any GoHighLevel account that has admin access and the ability to create Private Integrations. That includes Agency and Sub-Account plans.

What if it does not work for me?
Message us in the community. We will walk through the setup with you and make sure it works. If we cannot get it running, we will make it right.

Is this a subscription?
No. One payment, lifetime access. You also get every future update to the prompt library and agent configs at no extra cost.

Can I use this for my clients' GHL accounts?
Yes. Create a separate Private Integration Token for each client account and use the same agent configs with their details filled in.
