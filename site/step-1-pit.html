Step 1: Create Your Private Integration Token

A Private Integration Token (PIT) is like a secure key that lets Claude access your GHL account. This is the first thing you need to set up.

It takes about 3 minutes.

Open Private Integrations

  1. Log in to your GoHighLevel account
  2. Click Settings in the left sidebar
  3. Click Integrations
  4. Click Private Integrations

What you should see: A page listing any existing private integrations. It may be empty if this is your first one.

Create a New Integration

  1. Click the Create New button
  2. In the Name field, type: MCP Server Integration
  3. Leave the Webhook URL field blank
  4. Do not change any other default settings

What you should see: A form with a name field, webhook URL field, and a scopes section below.

Select Your Scopes

Scopes control what Claude is allowed to access. Select these scopes to cover the most common workflows:

ScopeWhat It Allows
contacts.readonlyRead contact information
contacts.writeCreate, update, and tag contacts
conversations.readonlyRead message history
conversations.writeSend messages and manage threads
opportunities.readonlyView pipeline deals
opportunities.writeCreate deals and move stages
calendars.readonlyCheck calendar availability
calendars.writeBook and manage appointments
locations.readonlyAccess account/location settings
workflows.readonlyView available workflows
users.readonlyView team member information

Find each scope in the list and check the box next to it.

What you should see: 11 scopes checked. The rest should remain unchecked.

Only enable the scopes listed above. Adding unnecessary scopes gives Claude broader access than needed. You can always add more later.

Save and Copy Your Token

  1. Click Save at the bottom of the page
  2. Your Private Integration Token appears on screen
  3. Copy it immediately and paste it somewhere safe
You will only see this token once. If you lose it, you will need to create a new integration. Copy it now and store it in a password manager or secure note.

What you should see: A long string of letters and numbers. This is your PIT token.

Quick Checklist

Before moving on, confirm:

  • Integration named MCP Server Integration
  • 11 scopes selected (contacts, conversations, opportunities, calendars, locations, workflows, users)
  • Token copied and saved somewhere secure
  • Webhook URL left blank
